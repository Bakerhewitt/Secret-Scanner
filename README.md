# Secret Scanner
Baker Hewitt

## Overview

**Secret Scanner** is a regex scanner that mines for "secrets" within an inputted file or directory.

## Features
- **Dual Scan Modes**: Can scan either a single file or an entire directory recursively.
- **Regex-based Detection**: Uses pattern-matching to find "secrets" such as passwords, API keys, and tokens from Datadog, Square, and Twilio.
- **Reporting**: Prints findings with filename, line number, secret type, and matched string.
- **CLI Interface**: Can be run interactively.
- **Logging**: Tracks scan activity.

## Requirements

Python Interpreter
